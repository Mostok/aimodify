A) Install - 0 steap - top

To run the script, we recommend using a VDS server or a dedicated server.

Requirements:

  • PHP >= 5.6.4
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Composer

B) Install - 1 steap - top

Copy/Upload the folder and files to your web server using cPanel or FTP

Install Laravel components with composer
# composer update

Laravel postinstall
# php artisan key:generate
# php artisan storage:link

C) Install - 2 steap - top

Create mysql database, database user and grant all permission to user for this database.

D) Configuration - top

All settings are in the .env file

APP_NAME - You Application Name
Example: APP_NAME="Tools"

APP_ENV - Status of your application (local - for development, production - Production)
Example: APP_ENV=local

APP_DEBUG - Debug mode (true or false)
Example: APP_ENV=true

APP_URL - Full url
Example: APP_URL=http://donations.klybok.net/

DB_* - SQL Server Configuration
Manual: https://laravel.com/docs/5.4/database
